  Setup Instructions

How to set up OneDrive file sharing for Chat:

1. Go to OneDrive and create a shared folder for your team.

2. Right-click the folder → Share → Set permissions to "Anyone with the link can edit".

3. Copy the share link and paste it in the field above.

4. Users in Chat can open this folder to upload files, then share the file's direct link in chat messages.

  Tip: For fastest downloads, users should share the direct download link of individual files from OneDrive.
